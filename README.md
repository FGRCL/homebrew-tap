# Fgrcl Tap

## How do I install these formulae?

`brew install fgrcl/tap/<formula>`

Or `brew tap fgrcl/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "fgrcl/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
